Gem Shooter
created by Adriaan Bosch for AUT COMP710

Controls

To nivagate the main menu use the "w" & "s" keys to select "new game" or "quit"
Use the key "e" to comfirm the selection.

Once in the game the controls consist of aiming "a" left and "d" right, "space" shoots the current gem towards the aim line.
To switch between the two gems given "q" does this.
Pressing "esc" will bring up the pause menu  which the user can select like the main menu to start a "new game" or "quit"
